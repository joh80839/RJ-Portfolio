Apple Website Edit

I modified Apple's homepage using Chrome DevTools. I changed the main title of the homepage, also known as a headline in html, and I added a few pink background elements in CSS, such as the Apple logo in the corner of the webpage, top bar, and lower bar above IPhone Air to pink. I wanted to change the whole black background behind the phone to pink, but there were limitations; a lot of boxes were tied to each other, which limits the ability for one edited box to make visual changes on the webpage. In a sense, if you edit one box then you have to edit many other boxes for changes to happen. These limitations helped me understand how complex websites can be.
